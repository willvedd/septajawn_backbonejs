Technical Details

Septajawn is built upon the Backbone.js framework and utilizes the default dependencies (Underscore, jQuery). All the data is pulled from a custom made JSON file.

Aside from the project's altruistic goals, performance was the primary consideration during development. The idea being, the faster and more lightweight the application, the more devices that could adequately run it, thus fulfilling the accessiblity goals. Originally the project was built without a framework or particular organization. Despite the application functioning as needed, the performance wasn't deemed adequate. Thus, the app was re-written from the ground up in Backbone. The perfomance gains from the reconstruction were substantial.
